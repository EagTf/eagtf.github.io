Commands Description
Play (P) Plays audio from YouTube or SoundCloud.
Playlist (Pl) Play a playlist from youtube
Pause Pause the currently playing music
Skip (S) Skip the currently playing song
Skipto (St) Skip to the selected queue number
Stop Stops the music
Shuffle Shuffle queue
Search Search and select videos to play
Resume (R) Resume currently playing music
Remove (Rm) Remove song from the queue
Queue (Q) Show the music queue and now playing
Np Show now playing song
Move (Mv) Move songs around in the queue
Lyrics (Ly) Get lyrics for the currently playing song
Loop Toggle music loop
Volume (v) Change volume of currently playing music
Clip Plays a clip sound
Clips List all clips
Kick Kicks a specific user
Ban Ban a specific user
Mute Mutes a specific user
Unmute Unmutes the user
Lock locks the channel
Unlock Unlocks the channel
Pruning Toggle pruning of bot messages
Serverinfo Shows the server information
userinfo Shows information for a specific user
Say Repeats the message you send
UpTime Check the uptime for the bot
Stats Shows some stats for the bot
hoststats Shows server statistics
Ping Shows the bots ping
Invite Send bot invite link
8ball Ask 8ball
Urban Search a word on Urban
Weather Shows weather of a city
Morse Encode or decode morse
Coinflip Flip a coin for a chance to get heads or tails!
Rps Play Rock Paper scissors with the bot!
Cat Sends a random image of a cat.
Dog Sends a random image of a dog.
Help Help Command
